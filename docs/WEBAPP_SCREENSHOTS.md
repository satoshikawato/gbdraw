# Web App Screenshot Guide

This guide standardizes screenshot names, captions, and callouts for the web app.

## Naming Convention

- Pattern: `webapp_<nn>_<slug>.png`
- Use two-digit numbers for ordering (01, 02, 03).
- Use lowercase and underscores for the slug.
- Keep the same viewport size and zoom for all images.

Examples:
- `webapp_01_overview.png`
- `webapp_02_inputs.png`
- `webapp_07_legend_editor.png`

## Shot List and Captions

## Callout Text Library

- Mode toggle
- Save Config
- Load Config
- Input files
- Track layout
- Show labels
- Palette selector
- Specific rules
- Generate Diagram
- Result preview
- DPI selector
- SVG/PNG/PDF export
- Legend Editor
- Canvas padding
- Drag to reposition
- Reset positions

## Alt Text Template

Use short, concrete descriptions:
`Alt: gbdraw web app showing <feature> with <action>.`
