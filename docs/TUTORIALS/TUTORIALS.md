# Tutorials

Step-by-step guides for the main `gbdraw` workflows.

## Introductory path

- [Tutorial 1: Customize a circular plot](./1_Customizing_Plots.md)
  Learn palettes, circular track layout, centered definitions, titles, and labels.
- [Tutorial 2: Compare genomes with BLAST and protein searches](./2_Comparative_Genomics.md)
  Build linear comparison plots, control record and region selection, and compare nucleotide with translated nucleotide similarity in circular LOSATN and TLOSATX rings.
- [Tutorial 3: Control colors, labels, and appearance](./3_Advanced_Customization.md)
  Use color tables, label filters, qualifier priority, manual label overrides, and styling options.

## More workflows

- [Tutorial 4: Protein comparisons from annotated CDS features](./4_Protein_Comparisons.md)
  Run pairwise, orthogroup, and collinear protein comparisons directly from annotated CDS features.
- [Tutorial 5: TSV manifests for CLI inputs](./5_Table_Driven_Inputs.md)
  Use row-coupled TSV manifests for records, conservation rings, and circular track slots.
- [Tutorial 6: Coverage depth and numeric tracks](./6_Depth_Quantitative_Tracks.md)
  Add depth tracks, axes, scaling, and GC content (%) tracks.
- [Tutorial 7: Linear layout, definitions, and rulers](./7_Linear_Layout.md)
  Customize linear track layout, rulers, titles, definition lines, and track slots.
- [Tutorial 8: Interactive SVG and saved sessions](./8_Interactive_SVG_Sessions.md)
  Write interactive SVG files and rebuild diagrams from saved sessions.
- [Tutorial 9: Feature visibility and shapes](./9_Feature_Visibility_Shapes.md)
  Override feature shapes and per-feature visibility.
