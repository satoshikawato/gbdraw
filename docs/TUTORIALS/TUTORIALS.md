# gbdraw command-line guides

These guides cover the main `gbdraw` command-line workflows.

## Circular diagrams and genome comparisons

- [Style a circular genome diagram](./1_Customizing_Plots.md)
  Set palettes, circular track layout, center labels, plot titles, and feature labels.
- [Draw genome comparison links from BLAST and protein searches](./2_Comparative_Genomics.md)
  Draw linear links from BLAST HSPs or CDS protein searches, select records or regions, and compare LOSATN with TLOSATX hits in circular rings.
- [Set feature colors and labels](./3_Advanced_Customization.md)
  Use color tables, label filters, qualifier priority, manual label overrides, and stroke or font settings.

## Protein searches, table inputs, and output controls

- [Draw protein matches from annotated CDS features](./4_Protein_Comparisons.md)
  Translate annotated CDS features, search the protein sequences, and draw pairwise matches, gbdraw similarity groups, or collinear blocks.
- [Use TSV manifests for CLI inputs](./5_Table_Driven_Inputs.md)
  Put record-specific inputs, BLAST ring sources, or circular track slots in TSV files.
- [Plot read depth and other numeric tracks](./6_Depth_Quantitative_Tracks.md)
  Plot per-base read depth, set quantitative axes and scaling, or show GC content as a percentage.
- [Arrange linear tracks, record labels, and rulers](./7_Linear_Layout.md)
  Position linear tracks, format record labels, and add titles, rulers, or custom track slots.
- [Create interactive SVGs and restore saved sessions](./8_Interactive_SVG_Sessions.md)
  Export standalone interactive SVG files and regenerate diagrams from saved sessions.
- [Control feature visibility and shapes](./9_Feature_Visibility_Shapes.md)
  Override feature shapes and per-feature visibility.
