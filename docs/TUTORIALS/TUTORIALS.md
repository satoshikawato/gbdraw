# Tutorials

Step-by-step guides for the main `gbdraw` workflows.

## Introductory Path

- [Tutorial 1: Customizing Your Plot](./1_Customizing_Plots.md)
  Learn palettes, circular track layout, centered definitions, titles, and labels.
- [Tutorial 2: Comparative Genomics](./2_Comparative_Genomics.md)
  Build linear comparison plots, control record/region selection, and contrast nucleotide- and translated-protein homology with circular LOSATN/TLOSATX rings.
- [Tutorial 3: Advanced Customization](./3_Advanced_Customization.md)
  Use color tables, label filters, qualifier priority, manual label overrides, and styling options.

## Advanced Workflows

- [Tutorial 4: Protein Comparisons Without Precomputed BLAST](./4_Protein_Comparisons.md)
  Run generated pairwise, orthogroup, and collinear protein workflows.
- [Tutorial 5: Table-Driven CLI Inputs](./5_Table_Driven_Inputs.md)
  Use row-coupled TSV manifests for records, conservation rings, and circular track slots.
- [Tutorial 6: Coverage Depth and Quantitative Tracks](./6_Depth_Quantitative_Tracks.md)
  Add depth tracks, axes, scaling, and GC percent tracks.
- [Tutorial 7: Linear Layout, Definitions, and Rulers](./7_Linear_Layout.md)
  Customize linear track layout, rulers, titles, definition lines, and track slots.
- [Tutorial 8: Interactive SVG and Session Round Trips](./8_Interactive_SVG_Sessions.md)
  Export standalone interactive SVG and regenerate diagrams from saved sessions.
- [Tutorial 9: Feature Visibility and Shapes](./9_Feature_Visibility_Shapes.md)
  Override feature shapes and per-feature visibility.
