gbdraw Web App Licensing and Attribution

This page documents the licenses and bundled third-party assets that ship with the static gbdraw/web/ application. The same directory is used for the hosted gbdraw.app deployment and for the local browser UI launched by gbdraw gui.

The hosted application is distributed as a static site on GitHub Pages behind the gbdraw.app custom domain. Pyodide and the browser-side assets listed below are vendored and served from the same origin, so the app does not need to fetch those runtime assets from third-party CDNs.

Back to gbdraw Project Repository Jump to License Texts

Distribution Notes

  • The web UI is a build-less single-page app loaded directly from index.html.
  • The GitHub Pages deployment copies the contents of gbdraw/web/ as static files.
  • Pyodide, jsPDF, DOMPurify, browser_wasi_shim, icons, and font assets are vendored into the repository.
  • This notices page is included in the shipped wheel so the same attribution is available from gbdraw gui.
